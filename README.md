# VILLA Fitness Clinic

Website oficial do VILLA Fitness Clinic, um estúdio de treino personalizado em Vila Verde.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Verificação

```bash
npm run lint
npm test
```

O conteúdo é escrito em português europeu e as fotografias utilizadas pertencem à seleção de
materiais do projeto. As imagens recolhidas de notícias permanecem fora do website até existir
confirmação de direitos de utilização.
